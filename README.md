# stay_reminded-bot

https://t.me/stay_reminded_bot

#### Бот, созданный для помощи школьникам и студентам в организации учебной деятельности. Напоминалки про дедлайны, хранилище ваших сертификатов, - всё это в одном чате Телеграмма.

__Требования:__
- python 3.12.0
- aiogram 2.23.1
- python-dotenv 1.0.0
